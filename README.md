# 🎯 Java Quiz Game

## 📌 Project Description
Java Quiz Game is a console-based application developed using Java. It allows users to answer multiple-choice questions, calculates the score, displays the percentage and grade, and provides an option to play the quiz again.

## ✨ Features
- 10 Multiple Choice Questions
- Instant Feedback (Correct/Wrong Answer)
- Displays Correct Answer
- Score Calculation
- Percentage Calculation
- Grade System
- Play Again Option
- User-Friendly Console Interface

## 🛠️ Technologies Used
- Java
- VS Code
- OOP (Object-Oriented Programming)
- Arrays
- Loops
- Conditional Statements
- Scanner Class

## ▶️ How to Run

1. Open the project in VS Code.
2. Compile the Java files.
3. Run `QuizGame.java`.
4. Press Enter to start the quiz.
5. Answer the questions by entering options (1–4).
6. View your score, percentage, and grade.

## 📸 Output

- Welcome Screen
- Quiz Questions
- Final Result
- Play Again Option
